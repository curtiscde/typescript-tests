# Fix The Tests

Simple typescript app with failing tests to fix

## Getting Started

 - Install VS Code
 - Install npm
 - Open Terminal
 - Clone this Repository
 - Install node modules: `npm install`
 - Run tests: `npm test`

## Unskipping Tests

To unskip a test, remove `.skip` from the `describe`:

```
describe.skip('test..') becomes describe('test..') 
```
